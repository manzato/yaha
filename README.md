# yaha
Yet Another Home Automation 


## Installing

```
$ sudo apt-get install mongodb
$ git clone git@github.com:manzato/yaha.git
$ cd yaha
$ sudo -H python setup.py install
```


## Development install

```
$ cd $clonedDir
$ sudo -H pip install -e .
```
